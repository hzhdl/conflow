# 实验设计

## 实验目的

- 构建跨链模拟器，来模拟跨链方案的执行，并测试其性能等指标。


## 实验内容

1. 测试真实发现算法的性能和实际的数据聚合效果。
2. 测试数据验证方案的性能和验证效果。


## 实验配置

- 整体对比：
    - Smartsync   迁移方案
    

- TD算法：
    - 数据源数量：100
    - 任务数量：1000
    - 作恶率：0.1
    - 拜占庭节点数量：10
    - Baseline:
      - CRH
- ZK验证算法：
    - Batchsize：1000
    - proofsize：10
    - delay：0.8
    - Gas overhead: 0.000001
    - Baseline:
      - 单次验证
      - HashChain数据结构
